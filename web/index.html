gap-data is an experiment in internet economics, to find a way to pay for shared services.

We'll start with a model, and then evolve it to support our costs.

Gap Data is open source.

Gap Data runs on gap-data

Gap Data is a web interface for appengine development.

Gap Data exposes its configuration to its site administrators, including all site content and programming.

Gap Data site administrators may adapt its web based programming to create groups or roles or capabilities' delegations to other web logons as desired.

Gap Data is like an "application wiki".

Sign in, find a friend or a project. Create a project.
