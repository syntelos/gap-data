
Run 'ant' with no argument to build all.  

Otherwise employ one of the following arguments.

    all      Build all subpackages
    odl      Generate Java from ODL in the GAE subpackage
    clean    Clean subpackages to initial state

    test     Run tests in subpackages
    run      Run development server in the GAE subpackage at http://localhost:8888/

    version  Print version of subpackages
    build    Increment version of subpackages
    major    Increment version of subpackages
    minor    Increment version of subpackages
